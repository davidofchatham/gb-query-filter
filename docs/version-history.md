# Version History Note

The upstream import (initial commit) shipped at **0.1.2** and already included: search, category/tag/extra-taxonomy filtering, checkbox + select control types, AJAX with full-reload fallback, Apply / auto-apply modes, Meta Box filtering, and GenerateBlocks 1.x + 2.0 hook support. A `targetId` attribute existed but was not yet wired to scoping.

All subsequent development — ACF integration, targeted mode, scoped URL parameters, class-based targeting, the `GBQF\Params` class, the expanded `Settings` class and admin page, the developer filters, and the AJAX/reset fixes — was consolidated into a single **0.2.0** release.

There was never a tagged 0.2.0 → 0.3.0 → 0.4.0 progression. Earlier drafts referenced fictional intermediate versions; treat **0.1.2 as the documented baseline and 0.2.0 as everything built on top of it.** Future releases bump from 0.2.0.

See [CHANGELOG.md](../CHANGELOG.md) for the per-version detail.
